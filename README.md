## Usage

Prerequisties: [https://bun.sh](https://bun.sh)

1. Clone repository
2. Fill .env using .env.sample
3. Fill backend/.env using backend/.env.sample
4. Start backend with `bun install && bun run dev`
5. Start frontend with `bun install && bun run dev`
